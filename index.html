CSS selectors and specificity

Basic CSS Selectors

Selector Description Syntax
Universal selects all elements of any type. (*)
Type matches the type of the element in the HTML document. elementname
Class matches elements based on the contents of their class attribute. .classname:
ID matches an element based on the value of the element's id attribute. #idname
Attribute matches elements based on the element having a given attribute explicitly set, with options for defining an attribute value or substring value match. [attr=value]

Specificity

Specificity is the algorithm used by browsers to determine the CSS declaration that is the most relevant to an element, which in turn, determines the property value to apply to the element. The specificity algorithm calculates the weight of a CSS selector to determine which rule from competing CSS declarations gets applied to an element.
